# Processamento de Percepção

- A inteligência é definida como a faculdade de conhecer, aprender, conceber e compreender, sendo o fator distintivo do ser humano frente aos demais animais.

## existem duas principais metodologias de processamento cognitivo aplicadas à resolução de problemas:

1. Top-Down -> Caracteriza-se pela observação prévia da "Big Picture" ou figura completa. Esta estratégia permite que o cérebro associe peças individuais ao contexto global, resultando em maior rapidez e eficiência sistêmica.

2. Bottom-Up -> Baseia-se na investigação empírica de peças individuais para descobrir conexões sem uma visão prévia do todo. É um processo de maior custo temporal e menor previsibilidade.

# Fundamentos de IA e Machine Learning

A Inteligência Artificial é consolidada como o conjunto de teorias e técnicas destinadas ao desenvolvimento de máquinas capazes de simular o comportamento inteligente humano.

1. Aprendizado: Definido como a capacidade de se adaptar, modificar e melhorar o comportamento e respostas. Existe uma simetria funcional entre o aprendizado humano e os algoritmos de Machine Learning

2. Machine Learning: Subcampo da IA que provê aos sistemas a habilidade de aprender a partir de dados e encontrar padrões ocultos sem a necessidade de programação explícita para tarefas específicas.

# Modelagem Matemática

O processo de aprendizagem automatizada é estruturado sobre três pré-requisitos: a existência de um padrão, a inexistência de um modelo matemático prévio que o explique e a disponibilidade de dados históricos.

A arquitetura do sistema é composta por:

- -> Input (x): Dados brutos do domínio (ex: dados de clientes).

- -> Output (y): Resultado da decisão ou classificação.

- -> Função Alvo (f:x→y): Representação ideal e desconhecida do relacionamento entre os dados.

- -> Espaço de Hipóteses: O algoritmo busca, dentro de um espaço de possibilidades, uma Hipótese Final (g) que apresente a maior proximidade possível com a função alvo.
