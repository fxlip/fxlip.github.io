chaves assimetricas = chave pública e privada

você compartilha a chave pública para que a mensagem seja criptografada com essa chave

para descriptografar é usado a chave privada

~/.ssh/id_rsa.pub
~/.ssh/id_rsa
~/.ssh/id_ed25519
~/.ssh/id_ed25519.pub

ssh_config (configurador de cliente)

sshd_config (configurador de servidor)

(conexão sem uso de senha)
ssh -i (seleciona a chave privada para fazer autenticação) -p (muda a porta) 
ssh-agent -> mostra variaveis de ambiente que o ssh-agent usa (SSH_AUTH_SOCK, SSH_AGENT_PID, SSH_AGENT_UID) para gerenciar as chaves e que pode permitir autenticação sem a necessidade de informar a senha a acada conexão.
ssh-add -> enviar as chaves para o agent
ssh-keygen -> gera as chaves
