# REVISÃO

---

[!] sudo

-> /etc/sudoers == visudo

ls /etc/sudoers.d/

% - configuração de grupo
# - não ativo

(pede a senha ou não)
usuario all=(all) NOPASSWD:/sbin/fdisk, PASSWD:/sbin/parted

--

outros comandos:
- last  (ultimos logins)
- lastb (ultimos logins bloqueados)

--

grep usuario /etc/shadow (informações da conta)

usuario:hash da senha:dias...

passwd - modifica, bloqueia ou desbloqueia a senha

chage -> faz modificações no usuário

- chage lock/unlock usuário
- chage expire usuário

--

pesquisas:

find -perm -4000 (USID) -2000 (SGID) -6000 (OS DOIS) -nouser -nogroup -type f

investigar conexões ativas no sistema, portas abertas

-> nmap -sS/sT/sA/sW/SM -sU | netstat -l | ss -listening | fuser | lsof -i (mostra todas conexões ativas)

--

ulimit (setar limites de usuario, tamanho de arquivos, numero máximo de arquivos abertos de um usuário, numero de cpus, quantos logins simultaneos) -a (ve os limites do usuario)

/etc/security/limits.conf
