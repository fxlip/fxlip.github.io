formato do /etc/passwd

nomeusuario:senha:idusuario:idgrupo:inscricao:home:bash

--

padrão de ids de usuários (ranges)

0 - root
1 a 99 - reservado para IDs pré-definidos
100 a 999 - reservado para contas administrativas e do sistema
1000+ -  usuarios

--

change pega e manipula as informações do shadow

--

usermod adiciona um usuario a um grupo

--

/etc/shadow armazena a senha do usuario em formato hash
