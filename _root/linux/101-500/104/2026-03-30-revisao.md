# REVISÃO

mkfs (é um frontend que chama outros comandos) 

padrão do mkfs é o ext2 (mais simples)

mkfs -j é ext3

porcentagem de espaço reservada para o usuário root em uma partição que usa ext4 é de 5%

mount exibe todos os sistemas de arquivos atualmente montados 

mount -a monta todos os pontos de montagem liostados no arquivo /etc/ftab, exceto o sque tenham a opção "noauto"

umask
novos arquivos ou novos diretorios?

0 (stickbit - especial)
0 leitura
2 escrita
2 grupo e outros

se compartilha o inode precisa estar na mesma partição

find -print é o padrão e o print0 remove os espaços em branco

mk2fs == *2fs

du = tamanho de diretórios e arquivos

df = tamanho e uso dos pontos de montagem

/etc/fstab

diferença entre link fisico e link simbólico

ln -s (simbólico)
ln -f (fisico)
