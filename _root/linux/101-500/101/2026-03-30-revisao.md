# REVISÃO

lspci -v
lspci -vv
lspci -vvv

modprobe - pelo nome
lsmod - caminho

BIOS == MBR (primeiros 512 ou 440 bytes + partições) -> bootloader -> kernel -> init -> systemd
UEFI == ESP (bootloader no boot/efi)

dmesg ou journalctl para ver logs

telinit (runlevel)

runlevel 0-6
0 desligamento
1 recover / singlemode / emergencia
6 reinicialização

define padrão no /etc/inittab

acpid - configurações de energia
