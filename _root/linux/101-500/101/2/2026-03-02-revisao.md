# REVISÃO

---

BIOS > MBR > Bootloader (GRUB2/LILO) > Kernel/Initramfs > Init (SysVinit/systemd)
- MBR -> primeiros 512 bytes do disco
- (Init)ram(fs) -> é uma estrutura de diretórios / montado em memória, para subir a máquina


-> MBR/DOS (tabela de particionamento)
- particoes primarias x estendidasx x logicas
- 4 particoes
- limite de 2tb

---

UEFI > Bootloader (GRUB) > Kernel > Init
- UEFI mais moderno, pula o MBR, busca direto no diretório
- Sempre acessa /boot/efi (FAT) <- onde está o bootloader (ESP - EFI System Partition)
- bootloader (GRUB) -> carrega o kernel (linux de fato) 
- Init (SysVinit/systemd) -> inicia redes, gráfico 

-> UEFI só consegue ler sistema de arquivos do tipo FAT

-> GPT (tabela de particionamento)
- 128 partições

---

tabela de particionamento != partição != filesystem (sistema de arquivos)

---

comandos úteis:

`#dmesg` -> kernel ring buffer
`#journalctl -b` -> log de boot
