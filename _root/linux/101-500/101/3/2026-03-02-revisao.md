# REVISÃO

---

-> sysV
- `/etc/inittab` Runlevels de 0 a 6 (0 é power off, 6 é roboot)
- `/etc/init.d/` onde ficam os scripts de inicialização dos serviços
- `service ssh stop`
- `/etc/rc*.d`
- `init` e `telinit` alterna entre os runlevels


-> Systemd
- Units -> definem os serviços, pontos de montagem, automount, dispositivos, agendamentos, sockets.
- Target -> Agrupa vários Units (Como se fosse um runlevel)
- `usr/lib/systemd` == `lib/systemd`
- -> systemctl 
- - list-units
- - stop service
