# REVISÃO

---

Entender como faz o mapeamento:

filesystem
/sys/ -> informações/drivers
`/proc/`
/dev/ -> gerenciado pelo udev (dispositovo)

Qual arquivo tem os IRQ? -> /proc/interrupts

cpuinfo (flags)

proc man info
 
---

módulos -> implementa coisas que não estão no kernel

- #insmod -> inserir modulo
- #rmmod -> remove modulo direto
- #modprobe -> gerencia módulos 
- #modinfo -> detalhes de módulos
- #lsmod -> lista todos os módulos
