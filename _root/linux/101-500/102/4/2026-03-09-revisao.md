# REVISÃO

---

dpkg - instala e remove pacotes dentro do debian
  --search -S   : Mostra o pacote que é dono do arquivo
  --listfile -L : Lista arquivos do pacote
  --purge -P    : Remove o pacote, incluindo os arquivos de configuração
  --remove -r   : Remove o pacote, mantendo os arquivos de configuração
  --verify -V   : Verifica a integridade do pacote
(diferenca entre purge e remove)

dpkg-reconfigure = solicita inputs do usuario

apt-get - gerenciador de pacotes que lida com dependencias e faz o download
apt - melhoramento gráfico do apt-get
  - update -> atualiza a base do que está disponível 
  - upgraade
  - dist-upgrade

[102.4] Qual opção do comando dpkg é utilizada para remover completamente os arquivos de um pacote, incluindo seus arquivos de configuração?

--delete
--remove
--purge
--clean

[102.4] Qual opção do comando apt-get irá atualizar para a versão mais recente todos os pacotes atualmente instalados no sistema, mas sem remover ou instalar nenhum novo pacote?

update 
upgrade X
dist-upgrade
full-upgrade
clean-upgrade
