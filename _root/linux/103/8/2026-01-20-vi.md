## Modo navegação

Para procurar por um conteúdo ->

- `/palavra` o cursor vai até a próxima ocorrência da palavra, do começo para o fim do arquivo.
- -> `[n]` vai para a próxima ocorrência da palavra. (do começo para o fim)
- -> `[N]` vai para a última ocorrência da palavra. (do fim para o começo)

Para procurar de forma inversa ->

- `?palavra`  o cursor vai até a próximo ocorrência da palavra, do final para o começo do arquivo.
- -> `[n]` vai para a próxima ocorrência da palavra. (do fim para o começo)
- -> `[N]` vai para a última ocorrência da palavra. (do começo para o fim)

Ferramentas -> 
- `[cc]` recorta a linha em que o cursor está e **ENTRA** no modo de edição
- -> `[p]` fora do modo de edição, cola a linha 
- `[dd]` recorta a linha em que o cursor está e **NÃO ENTRA** no modo de edição
- -> `[p]` fora do modo de edição, cola na linha de baixo
- `[d3d]` ou `[26dd]` recorta as próximas três linhas
- -> `[p]` fora do modo de edição, cola as próximas três linhas
- `[yy]` copia a linha em que o cursor está
- -> `[p]` fora do modo de edição, cola na linha de baixo
- `[y3y]` ou `[26yy]` copia as próximas três linhas
- -> `[p]` fora do modo de edição, cola as próximas três linhas

## Modo edição

- `[i]` para começar a inserir apartir do cursor
- `[a]` para começar a inserir apartir do próximo character
- `[A]` para começar a inserir no final linha
- `[o]` para começar a inserir na próxima linha
- `[O]` para começar a inserir na linha de cima
- `[ZZ]` para sair e salvar (Zalva e Zai)

## Modo comando

- `[:w]` para salvar e continuar no arquivo
- `[:q]` para sair
- `[:q!]` sair sem salvar
- `[:wq]` sair e salvar
- `[:x]` sair e salvar
- `[:!comando]` realiza qualquer comando no bash e volta para o arquivo
- `[:w nome-novo-arquivo]` ou `[:wq nome-novo-arquivo]` para salvar nomeando
- `[:e]` carrega nova versão do arquivo 
- `[:e!]` carrega nova versão do arquivo sobreescrevendo o atual

# NANO
- 

- `^` == `[ctrl]`
- `M` == `[alt]`
- `[alt]+[/]` == vai para o final do texto
- `[alt]+[\]` == vai para o começo do texto
- `[ctrl]+[Y]` == volta para a página anterior
- `[ctrl]+[V]` == vai para a próxima página
- `[ctrl]+[_]` == vai para linha,coluna
- `[ctrl]+[w]` == pesquisa

# EMACS
- 

- `[CTRL+x] + [CTRL+c]` -> sai do arquivo
- `[CTRL+x] + [CTRL+f]` -> abre ou cria um novo arquivo
- `[CTRL+x] + [CTRL+s]` -> salva o arquivo
- `[CTRL+x] + [s]` -> confirma se quer salvar
- `[CTRL+x] + [w]` -> salvar como
- `[CTRL+k]` -> recorta uma linha a partir do cursor
- `[CTRL+y]` -> cola tudo que estiver no buffer
- `[CTRL+/]` -> desfaz o último comando
- `[CTRL+e]` -> vai para o final da linha
- `[CTRL+a]` -> vai para o início da linha
- `[ALT+SHIFT+>]` -> vai para o final do buffer
- `[ALT+SHIFT+<]` -> vai para o começo do buffer
- `[ALT+x]` -> define tipo de extensão
