> **103.2**

---


`tail` mostra o final do arquivo (ultimas 10 linhas) (rabo)
- `tail -3` ou `tail -n3` mostra últimas 3 linhas

`head` mostra o começo do arquivo (cabeça)
- `head -3` ou `head -n3` mostra primeiras 3 linhas

---

`cat -A` mostrar characteres especiais

`cut -d` (define um limitador) (corte) [MUITO IMPORTANTE]

---

`nl` mostra o numero de linhas

`od` é pouco usado, para ver hexa decimal do arquivo ou algum caractere especial

- `split` é pouco usado mas cai mto, serve para separar um arquivo em vários
- -> por default separa arquivos em 1000 linhas

---

- `wc`
- -> Imprimi totais de novas linhas, palavras e bytes para cada FICHEIRO, e uma
linha de totais se mais de um FICHEIRO for especificado.

- `od` (octal dump)
- -> Escreve uma representação sem ambiguidade, em bytes octal por padrão,
de ARQUIVO para a saída padrão.
 
---

`sed` [CAI MUITO]

- não altera o conteúdo do arquivo
- pega o arquivo como entrada mas não altera o arquivo

- -> se quiser alterar, fazer uma saída para outro arquivo ">"
- -> ou usar `sed -i` para editar o arquivo de fato

- `sed 's/Palabra/Substitui' alunos.txt` (s/ procura)
- `sed 's/Procura/d' alunos.txt` (s/ procura e /d deleta)

---

- `tr` [CAI MUITO]
 -> troca um charactere por outro ou apaga characteres

- `tr` não aceita um arquivo como input
- -> precisa indicar o arquivo de entrada


`cat alunos.txt | tr A e` o **input do tr** é a **saída do cat**

-> como usa o TR para mudar a quebra de linha do padrão windows para o padrão linux em um arquivo? 

---

[103.2] Qual comando utilizado para exibir as últimas linhas de um arquivo? (Informe apenas o nome do comando, sem o caminho ou argumentos)
