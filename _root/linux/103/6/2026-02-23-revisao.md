# REVISÃO

---

-> O que conseguimos fazer na linha de comando é o nice. A priority é consequencia do ajuste do nice.

Nicenecess values rang from -20 (aumenta a prioridade) e a +19 (que diminui a prioridade do processo)

Só o root consegue aumentar a prioridade de um processo.

Testes para testar:

#nice -> inicia um processo com determinada prioridade
#renice / #top -> altera a prioridade de um processo
O valor default de nice é 0.

Iniciar um processo com nice, por padrão começa com 10.

nice -5 (5)
nice --5 (-5)

renice 5 (5)
renice -5 (-5)

-p pid (padrão)
-u usuario
-g grupo
