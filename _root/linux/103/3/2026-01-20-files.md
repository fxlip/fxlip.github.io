Começa sempre pela raíz (root) `/`

- Caminho absoluto: `/usr/local/bin`
- Caminho relativo: `~/local/bin`

---

- `.` é o diretório atual
- `pwd` mostra o diretório que está
- `cd ..` volta um diretório
- `cd ../..` volta dois diretórios
- `cd -` volta ao diretório que estava anteriormente
- `cd ~` vai para o diretório do usuário logado

---

- `ls` mostra os arquivos de um diretório
- `ls -a` mostra os arquivos de um diretório + ocultos
- `ls -l` lista detalhada dos arquivos e diretórios (data de última alteração)
- `ls -la` lista detalhada dos arquivos e diretórios + ocultos
- `ls -h` lista detalhada dos arquivos e diretórios com tamanho em Kb/Mb/Gb
- `ls -lR` recursivo, entra dentro de cada diretório e mostra o que tem dentro.

Detalhes:
`-rwxr-xr-x 1 fxlip fxlip   91 fev  1 01:32 Script_Exemplo.sh`

- `-rwxr-xr-x` determina o tipo e as permissões dos arquivos e diretórios
- `1` determina quantos hardlinks apontam para esse mesmo local
- `fxlip` nome do usuário que é dono do arquivo ou diretório
- `fxlip` nome do grupo que é dono do arquivo ou diretório
- `91` tamanho do arquivo em bytes
- `fev  1 01:32` data e hora da última alteração do arquivo
- `Script_Exemplo.sh` nome do arquivo

> CONJUNTO DE CARACTERES ESPECIAIS

- `ls -l Aula1*` mostra apenas os arquivos que começam com Aula1, inclusive o Aula1, e que tenham qualquer `*` sequencia depois disso
- `ls -l Aula1?` mostra apenas os arquivos que tenham qualquer characteres APÓS a `?`
- `ls -l Aula?0` mostra apenas os arquivos que tenham zero APÓS a `?`
- `ls -l Aula[123]` define uma lista de characteres para mostrar
- `ls -l Aula[!123]` define uma lista de characteres para excluir
- `ls -l Aula[1-5]` define um range de characteres para mostrar
- `ls -l Aula[!1-5]` define um range de characteres para excluir
- `ls -l Aula[1-5]?` define um range de characteres para mostrar com characteres depois
- `ls -l Aula{10,20,30}` define uma sequencia de strings específica
- `ls -l A{ula,ULA}1` define uma sequencia de strings específica

> COPIAR

- `cp -v` mostra detalhes (-v de verbose)
- `cp -i` impede confirmando se um arquivo pode ser sobrescrito

- `cp origem destino` copia o mesmo arquivo com outro nome
- - -> `cp Aula1 /tmp/CursoLinux1` cria um novo arquivo no tmp com o nome CursoLinux1
- - -> `cp Aula1 /tmp/` cria um novo arquivo no tmp com o nome Aula1

> AGRUPAR E COMPRIMIR ARQUIVOS #1

`tar` usado para agrupar e mover arquivos de um lugar para o outro
- `tar c` criar um arquivo agrupado .tar
- `tar x` extrair um arquivo agrupado .tar
- `tar t` listar arquivos que existem dentro de um arquivo compactado 
- `tar cf` cria um arquivo agrupado .tar e indica o nome do arquivo a ser criado
- `tar cpf` cria um arquivo agrupado .tar com nome definido preservando as permissões
- `tar cpvf` cria um arquivo agrupado .tar com nome definido preservando as permissões e vê os detalhes (verbose)
- `tar tf` listar arquivos que existem dentro de um arquivo compactado e exibe

-> para compactar se usam três principais programas de compactação de arquivos [-k keep=mantem]
- `gzip` exclui arquivo de origem e faz a compactação -> `gzip -k` mantem o arquivo de origem
- `bpzip2` exclui arquivo de origem e faz a compactação -> `bzip2 -k` mantem o arquivo de origem
- `xz` exclui arquivo de origem e faz a compactação -> `xz -k` mantem o arquivo de origem

-> agrupar + compactar 
- `tar zcpvf` cria um arquivo compactado com gzip (.tar.gz) com nome definido preservando as permissões e vê os detalhes (verbose)
- `tar jcpvf` cria um arquivo compactado com bzip2 (.tar.bz2) com nome definido preservando as permissões e vê os detalhes (verbose)
- `tar Jcpvf` cria um arquivo compactado com xz (.tar.xz) com nome definido preservando as permissões e vê os detalhes (verbose)

-> para descompactar [-d = descompress]
- `unzip` para arquivos do tipo zip
- `gzip -d` ou `gunzip` para arquivos .gz (ou tbz)
- `bzip2 -d` ou `bunzip2` para arquivos .bz2 (ou tgz)
- `xz -d` ou `unxz` para arquivos .xz (ou txz)

-> desagrupar + descompactar 

 `tar zxvf` desagrupa um arquivo, `z` [tipo de compactação gz], `x` [para extrair], `v` verbose e `f` para indicar o nome do arquivo

 `tar Jxvf` desagrupa um arquivo, `J` [tipo de compactação xz], `x` [para extrair], `v` verbose e `f` para indicar o nome do arquivo
