# // ecosystem

Em 2020, o impacto do documentário *The Social Dilemma* catalisou uma decisão que eu já amadurecia: o êxodo das redes sociais e o retorno ao controle dos meus dados.

Meu workflow deixa de ser apenas usar o computador e começa a ser sobre **resistência**.

### Operating Systems
Acredito em ferramentas certas para o trabalho certo, sem clubismo.

1.  **Debian GNU/Linux:** Meu sistema padrão. Se algo precisa funcionar sem falhas, roda aqui.
2.  **Trisquel:** Usado para estudos de certificação e compreensão da filosofia do Software Livre.
3.  **macOS (Catalina/Legacy):** Mantido em ambiente controlado de Hackintosh só para garantir que minhas soluções funcionem para os usuários Apple mas sem dar dinheiro para o ecossistema fechado deles.

### Portable Linux (Handheld)
* **Device:** SteamDeck (Arch Linux based).
* **Uso:** Curiosamente, uma das melhores ferramentas para entender a camada de compatibilidade *Proton/Wine*. Permite levar um ambiente Linux completo no bolso, testando a convergência entre entretenimento e código aberto.
